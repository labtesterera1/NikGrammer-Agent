☁️ Cloudflare Worker Proxy

Host your API keys on a Cloudflare Worker — no keys stored on device. Deploy worker.js to Cloudflare and paste the URL below.

📋 Worker Setup:
1. Go to dash.cloudflare.com → Workers & Pages → Create
2. Paste the worker.js code (provided in download)
3. Add secrets: GROQ_KEY, CEREBRAS_KEY, GEMINI_KEY
4. Copy your worker URL and paste it above
